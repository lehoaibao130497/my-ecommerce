# My E-Commerce (Week 1)

Project Fullstack Node.js + TypeScript đơn giản.

## Đã làm được gì
- Server Express + TypeScript chạy tại `http://localhost:3000`
- CRUD sản phẩm đầy đủ:
  - GET /products → xem danh sách
  - POST /products → thêm sản phẩm mới
  - DELETE /products/:id → xóa
  - PUT /products/:id → sửa
- Dữ liệu lưu tạm trong RAM (sẽ chuyển sang MongoDB tuần sau)

## Cách chạy
```bash
npm install
npm run dev
